Jak posortowa list zawierajc bardziej skomplikowane dane?

Metoda sort oraz funkcja wbudowana sorted posiadaj dodatkowy parametr - key. Przy pomocy tego parametru mona przekazywa klucz (w postaci obiektu funkcyjnego) zgodnie z ktrym ma si odby sortowanie. Funkcja przekazywana przy pomocy parametru key powinna zwraca warto typw ktre metoda sort i funkcja sorted s ju w stanie posortowa domylnie (int, float, str, ...).

In[4]:
data = [("kot", 2), ("pies", 87), ("kanarek", 21), ("rybki", 0)]

# Sortowanie po drugim elemencie krotki:
data.sort(key=lambda x: x[1])

print(data)

[('rybki', 0), ('kot', 2), ('kanarek', 21), ('pies', 87)]

Klasy:

Co to jest klasa (w Pythonie)?

  • zdefiniowany przez programist nowy typ - wyspecjalizowany pojemnik na dane,
  • klasa moe przechowywa dane i mie metody (ktre na tych danych pracuj),
  • wewntrz klasy tworzy si nowa przestrze nazw - nie wszystko co zdefiniowane jest wewntrz klasy musi (ale moe) by widoczne "na zewntrz",
  • na podstawie klasy mona tworzy obiekty tej klasy

Atrybuty publiczne i prywante klasy:

Pewne atrybuty klasy s widoczne na zewntrz, s to:

  • zmienne zdefiniowane w gwnej przestrzeni nazw klasy (tak jak w pierwszym przykadzie z klas i zmiennymi),
  • zmienne zdefiniowane nad self,
  • metody, czyli funkcje zdefiniowane w ciele klasy.

Elementy klasy mona ukrywa. Aby tego dokona nazw elementu naley rozpocz od znaku "_".

        self.surname = surname

        self._cumulative_len()

class Person:
    def __init__(self, name, surname, age):
        self.name = name
        self.surname = surname
        self.age = age

class List:
    def __init__(self, values):

Metoda specjalna __ add __ - dodawanie obiektw klasy:

In[17]:
class Vec2D:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    # Metoda specjalna definiujca regu dodawania.
    # Metoda ta zawsze powinna bra argument w posta drugiego obiektu -
    # tego ktry bdzie zumowany z self:
    def __add__(self, other):
        # Metoda powinna zwraca nowy obiekt (wynik):
        return Vec2D(self.x + other.x, self.y + other.y)

    def __repr__(self):
        return "[{}, {}]".format(self.x, self.y)

a = Vec2D(1, 2)
b = Vec2D(3, 4)

# Znak + dziaa teraz tak jak dla normalnych liczb:
c = a + b

print(c)

[4, 6]
